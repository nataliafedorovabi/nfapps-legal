1. Overview

Commission Tracker is a monday.com marketplace app that automatically calculates sales commissions based on your board data. It works with any board that contains deal/sales information — not just CRM boards. You map your columns once, and the app handles the rest.

The app provides four integrated features:

  • Board View — the main hub for creating commission plans and viewing calculated commissions in a table.
  • Dashboard Widget — KPI cards and per-rep performance overview.
  • Item View — a "what if" calculator that shows how much commission a deal is worth.
  • CSV Export — download summary or detailed commission reports.

2. Installation

  1. Go to the monday.com Marketplace and search for Commission Tracker.
  2. Click Install and follow the prompts to add it to your account.
  3. Open any board that contains your deals or sales data.
  4. Add the Board View: click the "+" button on the views panel, then select "Commission Tracker View".
  5. Add the Dashboard Widget: go to any Dashboard, click Add Widget, and select "Commission Dashboard".
  6. Add the Item View: click on any item, then click Add View and select "Commission Calculator".
Tip: You only need to add each view once per board or dashboard. After that, it will appear in your views panel automatically.

3. Getting Started — Creating Your First Plan

After adding the Board View, follow these steps to set up your first commission plan:

  1. Click "New Plan" to create a commission plan.
  2. Enter a plan name (e.g., "Q1 2026 Sales Plan").
  3. Select the plan type: Percentage, Fixed Amount, or Tiered.
  4. Set the rate, amount, or tier ranges depending on the plan type.
  5. Map your board columns:
    Mapping Description Required
    Deal Value Numbers column containing deal amounts Yes
    Deal Status Status column — select which label means "closed/won" Yes
    Sales Rep People column with the assigned salesperson Yes
    Close Date Date column for filtering by time period No
  6. Select the currency (USD, EUR, GBP, ILS, or RUB).
  7. Choose who this plan applies to:
    Option Description Available on
    All reps on this board Commission is calculated for every rep on the board (default) All plans
    Specific reps Select individual team members — only their deals count Starter
    Deals matching condition Filter by column value (e.g., Deal Type = "Enterprise") Pro
    Priority: If multiple plans exist on the same board, more specific plans take priority. Deals matched by a "condition" or "specific reps" plan will not be double-counted by an "all reps" plan.
  8. Click Save. Commissions will be calculated automatically.
Note: The app works with any board, not just monday CRM. As long as your board has the required columns (deal value, status, and sales rep), you can calculate commissions on it.

4. Commission Types

Percentage

A flat percentage of the deal value is paid as commission.

Example: 10% rate on a $25,000 deal = $2,500 commission.

Fixed Amount

A fixed dollar amount is paid for each closed deal, regardless of the deal size.

Example: $500 per deal. Close 8 deals = $4,000 commission.

Tiered Pro

Different commission rates apply based on deal value ranges. Higher-value deals earn higher rates, incentivizing reps to close bigger deals.

Example:

Deal Value Range Rate
Up to $50,000 5%
$50,001 — $100,000 8%
Above $100,000 12%

A $120,000 deal would earn: ($50,000 x 5%) + ($50,000 x 8%) + ($20,000 x 12%) = $2,500 + $4,000 + $2,400 = $8,900.

5. Board View — Commission Tracker View

The Board View is the main hub of Commission Tracker. It shows a commission table with all closed deals and their calculated commissions based on the active plan.

From the Board View you can:

  • Create and manage multiple commission plans.
  • Switch between plans to compare different commission structures.
  • View per-deal commission amounts in a table format.
  • See totals per sales rep.
  • Set up split commissions on individual deals.
  • Export data to CSV.

6. Dashboard Widget — Commission Dashboard

The Dashboard Widget provides a high-level overview of commission performance. Add it to any monday.com dashboard to see:

  • KPI Cards — total commissions paid, number of closed deals, average deal size, and top performer.
  • Per-Rep Performance Table — each sales rep's total deals, total deal value, and total commission earned.
Tip: The Dashboard Widget pulls data from the same commission plan configured in the Board View. Make sure you have at least one active plan set up first.

7. Item View — Commission Calculator

The Item View is a "what if" calculator that appears when you open an individual deal. It answers the question: "How much commission will I earn if I close this deal?"

The calculator shows:

  • The deal value from the mapped column.
  • The commission amount based on the active plan.
  • For tiered plans: a visual breakdown of which tiers apply and the rep's progress through the tier structure.

8. Split Commissions Starter+

Split commissions allow you to divide a deal's commission between multiple team members. This is useful when multiple reps collaborate on a sale.

  1. In the Board View, find the deal row you want to split.
  2. Click the split icon on that row.
  3. Add team members and set the split percentage for each person.
  4. Save the split. Each person will see their share reflected in the commission table.

Example: A $50,000 deal with 10% commission ($5,000 total). Split 60/40 between two reps: Rep A earns $3,000, Rep B earns $2,000.

9. CSV Export Starter+

Export your commission data for use in spreadsheets, accounting software, or payroll systems.

  1. In the Board View, click the "Export CSV" button.
  2. Choose the report type:
    • Summary — per-rep totals (rep name, number of deals, total deal value, total commission).
    • Detailed — per-deal breakdown (deal name, value, status, rep, commission amount, split info).
  3. The CSV file will download to your computer.

12. FAQ

Does the app work with any board?

Yes. Commission Tracker works with any monday.com board, not just CRM boards. As long as your board has columns for deal value, status, and sales rep, you can calculate commissions on it.

Are commissions calculated in real time?

Commissions are recalculated each time you open the Board View or refresh the page. Changes to deal values, statuses, or rep assignments are reflected immediately.

Can I have multiple commission plans on the same board?

Yes (Starter and above). You can create multiple plans on the same board and assign each to specific reps or filter by a column condition. The app uses a priority system to avoid double-counting: deals matched by a "condition" or "specific reps" plan won't be counted again by an "all reps" plan.

What currencies are supported?

USD, EUR, GBP, ILS, and RUB. Currency is set per plan and affects display formatting only — the app uses the raw number values from your board columns.

What happens if I uninstall the app?

All commission plan configurations and split settings stored by the app will be removed. Your board data is never modified or deleted by the app.
